# 🐾 PawPals Frontend

Frontend of the PawPals application built with **React** and **TypeScript**.  
It allows users to manage collaborators and mascots (dogs) with a clean and responsive UI.

---

## 🚀 Features

- User registration form
- View and manage collaborators
- Full CRUD for mascots
- Search mascots by name and age
- Responsive UI with Tailwind CSS
- API integration with backend

---

## 🛠️ Tech Stack

- React
- TypeScript
- Tailwind CSS
- Fetch API
