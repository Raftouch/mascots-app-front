🐾 PawPals Frontend

Frontend for the PawPals application built with React and TypeScript.

🚀 Features
User registration form
View and manage collaborators
CRUD for mascots (dogs)
Search functionality (name & age)
Responsive UI with Tailwind CSS
API integration with backend

🛠️ Tech Stack
React
TypeScript
Tailwind CSS
Fetch API
